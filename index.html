// what is sneak-up

Tactical Stealth
Combat

A browser-based 2v2 multiplayer shooter where two teams spawn in opposite corners and fight for control - all under fog of war. Your vision is limited. Every shadow is a threat. Every move is a risk worth taking.

Fog of War
Vision is limited to line-of-sight. Anticipate enemy positions, set ambushes, and exploit blind spots.
2v2 Teams
Coordinate with your teammate, split lanes, and apply coordinated pressure from multiple angles.
Multiple Maps
Each map has a distinct layout - open arenas, narrow corridors, maze-like structures. Pick your style.
Browser Native
No installer. No launcher. Open and play in any modern browser. Full gamepad support included.

// ai & bots

Code Your
Own Fighter

Bots in Sneak-Up are written in JavaScript with direct access to game state - enemy positions within line-of-sight, ally coordinates, map geometry, and more. Every tick, your function gets called and returns an action. It's that simple.

// built-in strategies
Two bots ship with the game: Default - a balanced companion that plays the map, and Attack - an aggressive bot that pushes and engages relentlessly.
// custom scripts
Write your bot directly in the in-game editor. Access enemy and ally positions, compute your own heuristics, and test it instantly in Bot 2v2 - all stored locally, up to 25 custom bots.
// ml models
Upload a trained .onnx model as your bot's brain. The server runs inference at game tick frequency using ONNX Runtime. Read how to train and export a model →
// resources
The bot API reference, simulation environment docs, and a PPO training baseline are available in the docs →
my_bot.js
// called every game tick function decide(obs) { const { me, enemies, allies, map } = obs; // find nearest visible enemy const target = enemies .filter(e => e.visible) .sort((a, b) => dist(me, a) - dist(me, b) )[0]; if (target) { return { move: toward(me, target), shoot: true, aim: angle(me, target), }; } // fall back to patrol return { move: patrol(me, map), }; }
